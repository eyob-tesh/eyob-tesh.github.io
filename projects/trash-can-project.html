Raspberry Pi-powered system that detects full bins, sends alerts, and tracks who empties the trash

Overview

Smart Waste is an IoT-enabled trash can that uses an HC-SR04 ultrasonic sensor on a Raspberry Pi to monitor fill levels. When the can remains full for more than ten seconds, the system sends SMS alerts via Gmail's SMTP service and plays an audible alarm using Pygame. NFC tags paired with a Flask application log which household member emptied the trash in a SQLAlchemy database, promoting accountability.

Image and video demonstration coming soon.

Technologies Used

  • Raspberry Pi
  • Python (RPi.GPIO, Pygame)
  • HC-SR04 ultrasonic sensor
  • Flask & SQLAlchemy
  • SMTP for SMS notifications
  • NFC tags

Key Features

  • Real-time trash level monitoring with ultrasonic sensing
  • Automatic SMS notifications when the bin is full
  • Audible alarm to alert nearby users
  • NFC tag integration to record who empties the trash
  • Database-backed log of disposal events
